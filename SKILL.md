# 济南公交查询

查询济南市公交车的实时位置和到站信息。

## 快速使用

```
用户: K93路公交到哪了？
→ 返回 K93 所有运行车辆的位置

用户: 我在新泺大街舜旺路，K163最近的车多久到？
→ 返回距该站最近的来车信息

用户: K163都有哪些站？
→ 返回 K163 全部站点列表
```

## 功能

| 功能 | 命令 |
|------|------|
| 查询公交实时位置 | `query.py 线路名` |
| 查询来车时间 | `query.py 线路名 --station 站名` |
| 查询线路站点 | `query.py 线路名 --stops` |
| 扫码登录 | `login.py` |

## 脚本

- **scripts/login.py** - 扫码登录，获取 token
- **scripts/query.py** - 查询公交信息
